HTML知识梳理(上)
作者：一件白衬衫

超级文本标记语言是标准通用标记语言下的一个应用，也是一种规范，一种标准，它通过标记符号来标记要显示的网页中的各个部分。 网页文件本身是一种文本文件，通过在文本文件中添加标记符，可以告诉浏览器如何显示其中的内容。浏览器按顺序阅读网页文件，然后根据标记符解释和显示其标记的内容，对书写出错的标记将不指出其错误， 且不停止其解释执行过程，编制者只能通过显示效果来分析出错原因和出错部位....

HTML知识梳理(下)
作者：一件白衬衫

HTML是用来描述网页的一种语言。

- HTML 指的是超文本标记语言 (Hyper Text Markup Language)

- HTML 不是一种编程语言，而是一种标记语言 (markup language)

- 标记语言是一套标记标签 (markup tag)

- HTML 使用标记标签来描述网页
